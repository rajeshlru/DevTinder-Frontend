## ğŸš€ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites
-   **Node.js** (v16 or later)
-   **npm** or **yarn**
-   A running instance of the [DevTinder Backend](https://github.com/rajeshlru/Devtinder-Backend)

### Installation
1.  **Clone the repository**
    ```sh
    git clone https://github.com/rajeshlru/DevTinder-Frontend.git
    ```
2.  **Navigate to the project directory**
    ```sh
    cd DevTinder-Frontend
    ```
3.  **Install NPM packages**
    ```sh
    npm install
    ```
4.  **Set up environment variables**
    Create a `.env.local` file in the root and add the backend API URL:
    ```env
    VITE_API_BASE_URL=http://localhost:5000/api
    ```
5.  **Start the development server**
    ```sh
    npm run dev
    ```
The application will be available at `http://localhost:5173`.

## ğŸŒŸ Core Architecture & Features

### ğŸ” The Authentication Flow
- **Secure by Design:** Utilizes JSON Web Tokens (JWT) for stateless, secure authentication.
- **Persistent Login:** Redux state persistence ensures users remain logged in across sessions for a smooth experience.
- **Route Protection:** A higher-order component guards protected routes, redirecting unauthenticated users automatically.

### â¤ï¸ The Swipe & Match Engine
- **Dynamic Profile Fetching:** The core feed intelligently fetches developer profiles from the backend API.
- **Optimistic UI Updates:** Swiping provides instant visual feedback, while the backend processes the action asynchronously.
- **Mutual Connection Logic:** A "match" is only created when both users swipe right on each other, triggering notifications and enabling chat.

### ğŸ’¬ The Real-Time Chat System
- **Powered by WebSockets:** A Socket.IO integration provides a low-latency, bidirectional communication channel.
- **Private Chat Rooms:** Once a connection is made, a unique chat room is created, ensuring conversations are private.
- **Instantaneous Feedback:** Features like typing indicators and message read receipts can be easily implemented.
